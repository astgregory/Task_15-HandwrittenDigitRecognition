# Классификация рукописных букв английского алфавита


Проект реализует нейронную сеть с полносвязными слоями для распознавания рукописных букв английского алфавита (A–Z). Модель достигает точности **99,24 %** на тестовой выборке.


## Задача

Обучить модель классификации рукописных букв, используя только полносвязанные (плотные) слои. Оптимизировать архитектуру сети (число слоёв и нейронов) для достижения максимальной точности. Визуализировать процесс обучения и сделать выводы на основе графиков.


## Используемые данные


**Датасет:** рукописные буквы английского алфавита (A–Z), 26 классов.  
**Формат:** CSV‑файл, каждая строка — одна буква.  
* Первый столбец — метка класса (0–25, где 0 = A, 1 = B, …, 25 = Z).  
* Столбцы 1–784 — пиксели изображения размером 28×28.  
**Размер выборки:** 297 960 образцов после разделения на обучающую и тестовую части (80 / 20 %).

## Архитектура модели

* **Тип:** полносвязная нейронная сеть (Dense).  
* **Слои:**
   * входной слой: 784 нейрона (28 × 28 пикселей);
   * скрытый слой: 456 нейронов, активация ReLU;
   * выходной слой: 26 нейронов (26 букв), активация Softmax.  
* **Оптимизатор:** Adam.  
* **Функция потерь:** categorical_crossentropy.  
* **Метрики:** accuracy, mae (средняя абсолютная ошибка).

## Параметры обучения

* **Эпохи:** 30.  
* **Размер батча:** 1528.  
* **Разделение данных:** 80 % — обучение, 20 % — тестирование, с перемешиванием.

## Результаты

* **Точность на тестовой выборке:** 99,24 %.  
* **Потери на тестовой выборке:** 0,0359.  

Модель демонстрирует высокую точность и отсутствие признаков переобучения: кривые обучения и валидации идут параллельно, без расхождения.

## Визуализация

В проекте построены следующие графики:

1. **График точности** (accuracy) — динамика точности на обучающей и тестовой выборках по эпохам.
2. **График потерь** (loss) — динамика функции потерь на обучающей и тестовой выборках.
3. **График средней абсолютной ошибки** (mae) — динамика MAE на обучающей и тестовой выборках.

Графики подтверждают стабильное обучение и хорошую обобщающую способность модели.

## Вывод

Модель успешно справляется с задачей классификации рукописных букв. Высокая точность (более 99 %) и плавное снижение потерь и ошибок говорят о корректной настройке гиперпараметров и отсутствии переобучения. Дальнейшая оптимизация может включать добавление дополнительных слоёв или нейронов для потенциального повышения точности.
